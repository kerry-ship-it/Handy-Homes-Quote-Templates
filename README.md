# Handy Homes Quote Templates

Shared customer-facing quote templates for Handy Homes.

## Current Templates

- `exterior-home-care/EHC Quote Template.html`
  - current local EHC quote template
  - copied from the latest working version in the website repo so this repo can be the long-term home for quote templates
- `handyman/handyman-proposal-template.html`
  - customer-facing handyman proposal template
- `gutter-cleaning/`
  - gutter quote flow pages
- `painting/`
  - placeholder for painting quote template work

## Branches

- `main`
  - shared baseline for the repo
- `codex/exterior-home-care-quote-template`
- `codex/handyman-quote`
- `codex/painting-quote`

## Notes

- This repo is intended to be the central place for Handy Homes quote templates.
- Salesforce or other internal systems may feed data into these templates, but this repo is where the customer-facing template files live.
- Local Vercel metadata and Finder files are ignored.
